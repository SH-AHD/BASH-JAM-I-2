# Video Audio Extractor Script

A Bash script that extracts audio from video files (MP4, MKV, AVI, MOV) either from a single file or an entire folder.  
It uses `ffmpeg` to convert video files to MP3 audio.

## Features

- Extract audio from a single video file
- Extract audio from all video files in a folder
- Option to rename output files
- Option to save extracted audio to a new folder or the same folder as the video
- Supports multiple Linux distributions and installs `ffmpeg` if missing


### How to Run

1.Give execution permission (only the first time):
chmod +x extract.sh

2. ./extract.sh
