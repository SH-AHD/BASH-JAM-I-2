# Guess Number Game

A simple number guessing game where the player tries to guess a randomly generated number between 1 and 1000.  
The goal is to find the number in as few attempts as possible.

## How to Play

1. Give execution permission** to the script (only the first time):
chmod +x guess-number.sh

2.Run the game using:
./guess-number.sh

The computer will choose a random number between 1 and 1000.
Enter your guesses when prompted.
The game will give hints:

"the value is smaller" → your guess is bigger than the number
"the value is bigger" → your guess is smaller than the number

The game tracks your attempts, so try to guess the number in as few tries as possible.
