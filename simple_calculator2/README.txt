# Simple Calculator

- The script takes 3 variables: num 1, num 2 and operator

_ The script only takes numbers and the following operators: +, -, *, /

_ In case of an invalid input the script will retake the last input

- Workflow: 
1. the user runs the script
2. user inputs num1
3. user inputs operator
4. user inputs num2
5. script output the resualt of the operation
6. user input yes to do other operation (the script will loop to point 1 in the workflow) or input no to close the script
