# Hangman / Guess the Word Game

A simple Bash script implementing a Hangman-style game in the terminal.  
Guess the letters of a randomly selected word before the hangman is fully drawn!

## Features

- Randomly selects a word from a predefined list of Linux-related terms
- Terminal-based Hangman game with ASCII art
- Tracks wrong guesses and displays the hangman figure
- Simple and interactive for learning bash scripting and terminal input

## How to Run

1.Give execution permission (first time only):
chmod +x hangman.sh

2. ./hangman.sh
